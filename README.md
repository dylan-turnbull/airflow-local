# Run Airflow Locally

## Prerequisites
1. Download K9s
    - MacOs  
        ```
        # Via Homebrew
        brew install derailed/k9s/k9s
        # Via MacPort
        sudo port install k9s
        ```
    - Linux
        ```
        # Via LinuxBrew
        brew install derailed/k9s/k9s
        # Via PacMan
        pacman -S k9s
        ```
    - Windows  
        ```
        # Via scoop
        scoop install k9s
        # Via chocolatey
        choco install k9s
        ```
2. Clone this repository
    - `git clone yada-yada-yada`

path: "{HOME_DIR}/airflow-dags"
